# font-size

指定字体的大小。因为该属性的值会被用于计算 em 和 ex 长度单位，定义该值可能改变其他元素的大小。
一个流行的技巧是设置 body 元素的字体大小为 62.5%（即默认大小 16px 的 62.5%），等于 10px。现在你可以通过计算基准大小 10px 的倍数，在任何元素上方便使用 em 单位。这样 6px = 0.6em,8px = 0.8em。

# line-height vertical-align

行高，两行文字基线之间的距离。
行高 = 内容区宽度 + 行间距。行间距一分为 2，将其一半分别应用到内容区的顶部和底部

# line-height 的实战应用

  1. 大小不固定的图片、多行文字垂直居中
  2. 不定尺寸图片或多行文字的垂直居中
  • 主体元素 liine-block 化
  • 0 宽度，100% 高度辅助元素
  • vertical-align:middle

在使图片垂直居中的时候，需要让中线和基线合并，否则因为基线对齐的问题，图片会略微向上偏移

# 面试题：img 默认下间距解决方法

  1. display:block (可以导致 vertical-align:baseline 失效)
  2. vertical-align:top/bottom/middle（非 baseline 即可）
  3. line-height 足够小（让图片和父元素贴近，这样基线对齐的 x 就会和图片的下边重合）
  4. font-size:0
    以上所有的属性均添加给 img
