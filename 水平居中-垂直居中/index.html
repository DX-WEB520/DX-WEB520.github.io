# 水平居中

# 文字

text-align:center (只作用域文本元素)

# 块元素

margin: 上下任意值 左右 auto,auto 只对于水平起作用，上下不起作用。

# 多个块级元素

(例如：轮播图中的不定个数小圆点水平居中效果的实现)

# 方法一 : inline-block + margin

# 方法二：定位方法 + margin 负值

# 方法三:flex

# 垂直居中

# 文字

line-height:height

# 块元素

  1. 定位 + margin 负值 : top:50%; + margin-top:-200px;
  2. 定位 + transform : top:50%; + transform:translateY (-50%)
  3. top:0;bottom:0;margin:auto
  4. 给父元素设置 flex : display:flex;align-items:center;
